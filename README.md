# IoT Experiments Repository

This repository contains **12 IoT experiments** developed for learning and educational purposes. The projects demonstrate various IoT concepts, sensor interfacing, wireless communication, and microcontroller programming. Each experiment is organized in a separate file or folder for easy access and understanding.

Students can use this repository as a reference for practical IoT implementation, academic projects, and skill development. The code is designed to be simple, well-structured, and easy to modify for further experimentation and learning.
